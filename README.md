# Palavra Viva — Leitura Bíblica Reformada

PWA de leitura bíblica diária com os padrões doutrinários da fé reformada.

## Como fazer deploy na Vercel (gratuito)

### Opção 1 — Via GitHub (recomendado)

1. Crie uma conta gratuita em [github.com](https://github.com)
2. Crie um novo repositório (ex: `palavra-viva`)
3. Faça upload de todos os arquivos desta pasta
4. Acesse [vercel.com](https://vercel.com) e faça login com sua conta GitHub
5. Clique em "Add New Project" → selecione seu repositório
6. Clique em "Deploy" — em 30 segundos o app estará no ar!
7. Você receberá um link tipo: `https://palavra-viva.vercel.app`

### Opção 2 — Via Vercel CLI

```bash
npm install -g vercel
cd pasta-do-projeto
vercel
```

## Como adicionar ao iPhone (tela inicial)

1. Abra o link do app no **Safari** (não Chrome)
2. Toque no ícone de **Compartilhar** (quadrado com seta ↑)
3. Role para baixo e toque em **"Adicionar à Tela de Início"**
4. Toque em **"Adicionar"** no canto superior direito
5. Pronto! O app aparecerá na sua tela inicial como um aplicativo normal

## Funcionalidades

- ✝ Plano de leitura de 365 dias (AT + NT em paralelo)
- 📖 Catecismo de Heidelberg (129 perguntas)
- 📖 Breve Catecismo de Westminster (107 perguntas)
- 📜 Cânones de Dort (5 artigos)
- 📜 Confissão de Fé de Westminster (capítulos principais)
- 📜 Confissão Belga (artigos principais)
- 🔥 Contador de sequência (streak) de leitura
- 📅 Calendário interativo com marcação de dias
- 📊 Progresso mensal e anual
- 💾 Funciona offline (PWA)
- 📲 Instalável no iPhone e Android
